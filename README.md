# 🚀 LearnPro: Advanced JavaScript Learning Ecosystem

LearnPro is a modern, interactive web platform designed to take users from JavaScript basics to advanced mastery. Unlike traditional tutorials, LearnPro combines a comprehensive 100-article encyclopedia with a real-time code playground and a community-driven problem-solving hub.

![LearnPro Preview](https://via.placeholder.com/800x400?text=LearnPro+Platform+Interface)

---

## 🌟 Key Features

### 1. 📖 JS Encyclopedia (Wiki)

A searchable database of over 100 JavaScript topics. Each article is loaded dynamically via JSON, ensuring lightning-fast performance and easy updates.

### 2. 💻 Interactive Coding Lab

A built-in code editor that allows users to write JavaScript and see immediate output in a virtual console. No local setup required!

### 3. 🏆 Gamified Quizzes

Test your knowledge with timed "Sprints." Earn points, track your score, and review detailed explanations for every wrong answer.

### 4. 👤 Evolution Profile

A dynamic user dashboard that tracks:

- **Mastery Percentage:** Visualized with progress bars.
- **Activity Feed:** A history of completed lessons.
- **Rankings:** Progress from "Novice" to "JS Master" based on quiz performance.

### 5. 🤝 Community Submissions

A mini-forum where students can post code snippets, describe bugs, and get help from the community.

---

## 🛠️ Technical Stack

- **Frontend:** HTML5, CSS3 (Tailwind CSS Framework)
- **Logic:** Vanilla JavaScript (ES6+)
- **Storage:** Browser `localStorage` (No database required for the demo)
- **Data:** Centralized JSON files for Lessons, Quizzes, and Wiki Articles.
